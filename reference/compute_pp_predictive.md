# Compute PP using predictive method (Monte Carlo)

Compute PP using predictive method (Monte Carlo)

## Usage

``` r
compute_pp_predictive(state, n_add, theta, base_args, scenario_params)
```

## Arguments

- state:

  Current trial state

- n_add:

  Additional patients per arm

- theta:

  Hybrid design parameters

- base_args:

  evolveTrial base configuration

- scenario_params:

  Scenario parameters

## Value

Predictive probability
