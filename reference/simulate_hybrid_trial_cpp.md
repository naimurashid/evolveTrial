# Simulate a single hybrid trial (C++ implementation)

Simulate a single hybrid trial (C++ implementation)

## Usage

``` r
simulate_hybrid_trial_cpp(theta_list, base_args_list, scenario_params_list)
```

## Arguments

- theta_list:

  List of design parameters

- base_args_list:

  List of base trial configuration

- scenario_params_list:

  List of scenario parameters

## Value

List with trial results
