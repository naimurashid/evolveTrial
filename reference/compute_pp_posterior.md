# Compute PP using posterior method (fast approximation)

Compute PP using posterior method (fast approximation)

## Usage

``` r
compute_pp_posterior(state, n_add, theta, base_args)
```

## Arguments

- state:

  Current trial state

- n_add:

  Additional patients per arm

- theta:

  Hybrid design parameters

- base_args:

  evolveTrial base configuration

## Value

Predictive probability
