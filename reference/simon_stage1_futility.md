# Check Simon-style stage 1 futility rule

In classical Simon design, stop at stage 1 if responses \<= r1

## Usage

``` r
simon_stage1_futility(n_responses, r1)
```

## Arguments

- n_responses:

  Number of responders at stage 1

- r1:

  Stage 1 futility boundary (stop if responses \<= r1)

## Value

TRUE if should stop for futility
