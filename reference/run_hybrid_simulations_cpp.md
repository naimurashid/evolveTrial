# Run multiple hybrid trial simulations (C++ implementation)

Run multiple hybrid trial simulations (C++ implementation)

## Usage

``` r
run_hybrid_simulations_cpp(
  n_sim,
  theta_list,
  base_args_list,
  scenario_params_list
)
```

## Arguments

- n_sim:

  Number of simulations

- theta_list:

  Design parameters

- base_args_list:

  Base configuration

- scenario_params_list:

  Scenario parameters

## Value

Data frame with simulation results
