# Compute between-arm posterior probability (C++)

Compute between-arm posterior probability (C++)

## Usage

``` r
compute_ba_posterior_cpp(a_exp, b_exp, a_ref, b_ref)
```

## Arguments

- a_exp:

  Posterior shape for experimental arm

- b_exp:

  Posterior rate for experimental arm

- a_ref:

  Posterior shape for reference arm

- b_ref:

  Posterior rate for reference arm

## Value

P(HR \< 1 \| data)
