# Compute predictive probability curve

Compute predictive probability curve

## Usage

``` r
compute_pp_curve(state, theta, base_args, scenario_params)
```

## Arguments

- state:

  Current trial state

- theta:

  Hybrid design parameters

- base_args:

  evolveTrial base configuration

- scenario_params:

  Scenario parameters

## Value

Data frame with n_add and pp columns
